About the project

Why GameVault exists

The idea

GameVault is a small database of video games, built to practice organizing real information — titles, genres, platforms, release dates, ratings and system requirements — into something people can actually search through and click into, instead of just reading a static list.

Every title has its own full detail page with poster art, stats, and requirements, reached by clicking any card in the catalog.

Built with

  • HTML5 for structure and semantics
  • CSS3 for the dark gaming-platform design
  • Vanilla JavaScript for search, filtering and the detail pages
  • Hosted for free on GitHub Pages

How the data works

All 12 titles live in one JavaScript array inside script.js, each with its poster image, publisher, release date, and requirements. The homepage spotlights the top-rated title, the Browse page filters the whole catalog live, and every card links to game.html?id=..., which reads that ID from the URL and renders the matching title's full page — one template, twelve pages.
