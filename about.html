About the project

Why GameVault exists

The idea

GameVault is a small database of video games, built to practice organizing real information — titles, genres, platforms, release years and ratings — into something people can actually search through, instead of just reading a static list.

Every game is shown as a "cartridge" card. Clicking one opens a full stat sheet with the publisher, release year, platform and player count, styled after old game cartridge labels.

Built with

  • HTML5 for structure and semantics
  • CSS3 for the retro-arcade visual design
  • Vanilla JavaScript for search, filtering and the details modal
  • Hosted for free on GitHub Pages

How the data works

All 12 titles live in one JavaScript array inside js/script.js. The homepage pulls the four highest-rated entries, and the Browse page renders every entry and filters it live as you type or change the dropdowns — no page reloads, no backend server.
